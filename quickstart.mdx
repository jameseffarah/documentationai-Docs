## Prerequisites

Before starting, ensure you have:

<Callout kind="info">

- A modern web browser like Chrome, Firefox, or Safari
- An email address for account verification
- For teachers: School or organization email (recommended)
- Internet connection for real-time collaboration

</Callout>

## Choose Your Role and Sign Up

Llamaroo supports both teachers and students. Select your role to begin.

<Tabs>
  <Tab title="Teacher" icon="users">
    Teachers build interactive lessons with gamification elements.

    <Steps>
      <Step title="Visit the Site" icon="globe">
        Go to [https://llamaroo.com](https://llamaroo.com) and click **Start free**.
      </Step>
      <Step title="Create Account" icon="user-plus">
        Enter your email, create a password, and select **Teacher** role.
      </Step>
      <Step title="Verify Email" icon="mail">
        Check your inbox and click the verification link.
      </Step>
    </Steps>
  </Tab>
  <Tab title="Student" icon="book-open">
    Students join classes and complete gamified lessons.

    <Steps>
      <Step title="Visit the Site" icon="globe">
        Go to [https://llamaroo.com](https://llamaroo.com) and click **Join a class**.
      </Step>
      <Step title="Enter Details" icon="edit">
        Provide your name, email, and class code (from your teacher).
      </Step>
      <Step title="Confirm" icon="check-circle">
        Agree to terms and access your dashboard.
      </Step>
    </Steps>
  </Tab>
</Tabs>

## Log In and Explore the Dashboard

Once signed up:

<Steps>
  <Step title="Log In" icon="log-in">
    Return to [https://app.llamaroo.com](https://app.llamaroo.com), enter your credentials.
  </Step>
  <Step title="Dashboard Tour" icon="layout">
    Review key sections:
    
    | Section       | Purpose                          |
    |---------------|----------------------------------|
    | My Classes    | Manage or join classes           |
    | Lessons       | Build or complete interactive content |
    | Analytics     | Track progress and engagement    |
    | AI Builder    | Generate lessons quickly         |
  </Step>
  <Step title="Customize Profile" icon="settings">
    Upload avatar and set grade level preferences (Pre-K to Doctoral).
  </Step>
</Steps>

<Callout kind="tip">
  Use the search bar to quickly find demo courses or shared lessons.
</Callout>

## Join an Existing Class

Students: Use your teacher's class code.

````jsx
// Example class code format (provided by teacher)
const classCode = "LLM-ABC123";
console.log(`Join using code: ${classCode}`);
````

<Steps>
  <Step title="Enter Code" icon="key">
    Navigate to **My Classes** > **Join Class** and input the code like `LLM-ABC123`.
  </Step>
  <Step title="Access Content" icon="play-circle">
    Browse available lessons and start gamified activities.
  </Step>
</Steps>

## Build Your First AI-Generated Lesson

Teachers: Create interactive content in minutes.

<Steps>
  <Step title="Start AI Builder" icon="zap">
    Go to **Lessons** > **New Lesson** > **AI Generate**.
  </Step>
  <Step title="Enter Prompt" icon="edit-3">
    Type: "Create a Pre-K lesson on animal sounds with quizzes and badges."
  </Step>
  <Step title="Review and Publish" icon="rocket">
    Edit gamification elements, then share with a class code.
  </Step>
</Steps>
