## Migration Notes

<Callout kind="alert">

Review these steps if upgrading from v0.x. No data loss occurs, but verify your custom themes.

</Callout>

<Steps>

<Step title="Backup Data" icon="database">

Export courses via dashboard: Settings > Export > JSON.

</Step>

<Step title="Update SDK" icon="download">

Install latest:

<CodeGroup tabs="npm,yarn">

````bash
npm install llamaroo-sdk@latest
````

````bash
yarn add llamaroo-sdk@latest
````

</CodeGroup>

</Step>

<Step title="Regenerate Tokens" icon="key">

Use new endpoint: `POST https://api.llamaroo.com/v1/auth/refresh`.

<ParamField header="Authorization" param-type="string" required="true">

`Bearer YOUR_TOKEN`

</ParamField>

</Step>

<Step title="Test Courses" icon="play-circle">

Preview all classes in staging mode.

</Step>

</Steps>
